Tras instalar node.js
https://nodejs.org/es

Para crear un proyecto ejecutamos
```
npx create-react-app my-app
cd my-app
npm start
```

Una vez tenemos esto podemos añadir la carpeta al vscode
y en el apartado de terminal abrir una nueva y ejecutar
```
npm install
```

instalar componentes nuevos
```
npm install --save styled-components
```
